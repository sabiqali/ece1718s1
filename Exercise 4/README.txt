1. There are 3 character device drivers included, to run them please replace their name in the Makefile and run "Make" and then insert the modules in the kernel.

2. Use gcc to compile part3 and part4 and make sure all 3 are inserted before running them. 

3. Please note that I have used sw, ledr, hex, key and not KEY,SW,HEX,LEDR. If the code used to test contains these name of files, it will throw and error.

4. The game involves multiplication and gets more difficult after 5 correct answers.
